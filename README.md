# Stack
- Vue 3 Composition API
- TypeScript
- Tailwind CSS
- Vite
- Vue Router
- Axios
- shadcn/vue for elements
- Node v22.17.0

# How to run project
Clone this project.


### Install dependencies
```bash
npm install
```

### Run server
```bash
npm run dev
```

Now you can visit http://localhost:5173
